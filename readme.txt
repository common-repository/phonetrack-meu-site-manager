Allows you to automatically replace the phone numbers of your site (website, landing page or third party) with the PhoneTrack number.

== Description ==

Integrating PhoneTrack with "Meu site" makes it possible to automatically replace the phone numbers of your pages (site, landing page or third parties) with the PhoneTrack number, without the need to change the code of the current website. This integration should be used when you have ads in AdWords, Bing, Facebook, and others, and you want to measure your leads.

[Access the integration page for more information](https://phonetrack.app/account/integracao/script)

== Installation ==

1. Upload the `phonetrack-meu-site-manager` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the integration Hash from PhoneTrack to the settings (Admin > Settings > PhoneTrack Meu Site Manager)
